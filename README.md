# Marketing Campaign Response Prediction

## 📌 Project Overview
This project predicts whether a customer will subscribe to a marketing campaign using Machine Learning (Random Forest Classifier).

## 📊 Dataset
- Source: Bank Marketing Dataset
- File Used: bank.xls
- Target Variable: Subscription (Yes/No)

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn

## 🧠 Machine Learning Model
- Random Forest Classifier
- Train-Test Split
- Accuracy Evaluation
- Confusion Matrix

## 📈 Results
The model achieved good prediction accuracy in identifying potential customers likely to subscribe.

## 📂 Files in Repository
- AIML_PROJECT1.ipynb → Jupyter Notebook with full code
- bank.xls → Dataset
- Marketing_Campaign_Prediction_8slides.pptx → Project Presentation

## 🚀 How to Run
1. Upload dataset in same directory
2. Open Jupyter Notebook
3. Run all cells
